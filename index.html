    Projects I've Worked On

    Banking REST API Microservices

    A banking REST API enables the creation of customers, which can create banking accounts. A transaction endpoint can be used to withdraw or deposit funds between accounts.

    To use the banking api, a banking authorization service must grant approval using jwt. The banking auth api has user login for both customer and admin roles. This means that certain endpoints from the banking api require a user to be logged in and with the appropriate access level.

    The project has been documented, tested and developed using a SDLC.

    Tools Used: Go, Gorilla, JWT, MySQL, Zap, Docker

    MetaTyping (keyboard) Application

    MetaTyping is a typing CLI application that displays words that the user types. The app helps users improve their typing speed by recording typing metrics like accuracy and speed. The app uses a columnar-vectorized database to store word metrics so problematic words can be recommended.

    MetaTyping has typing sessions through drills, word lists or custom text that show typing metrics after finishing a session.

    Users can keep track of their progression through a dashboard. It features line graphs that show an individual word or word group progression over time. Additionally, a data table/ bar plot shows words metrics like average speed and accuracy that can be filtered.

    Tools Used:Python, Curses, Pandas, Plotly-Dash, DuckDB

    Indeed Job applier

    The Indeed Job Applier helps perform the job apply process by web scraping specific job categories, filtering out jobs by tags, applying semi-automatically, and saving jobs applied to.

    The app uses a CLI interface with json tags to tailor the job postings scraped. These jobs are saved in an unprocessed job listings. The app then allow users to go through these postings which will automatically open 5 tabs linking to the job pages. The types in which job postings they applied too and the process repeats. The jobs applied to get sent to a applied to jobs list that can be accessed.

    Tools Used: Python, Scrapy, json
