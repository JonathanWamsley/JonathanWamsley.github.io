    Typing Application

    I got into touch typing and wanted features that current typing websites lack so I created a typing application

    I wanted to type content I find interesting that allows users to paste and save information, be able to navigate (backtrack or skip) text while you are typing, see real time typing speed, and store metrics.

    Being able to store individual word metrics in a database, I then created a dashboard to view strengths and weaknesses in typed words. The program then suggests problematic words the user.
